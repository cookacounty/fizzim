# Testbench Area

This folder holds pinned reference material used by the public regression flow.

## Legacy Fizzim Reference

`legacy/fizzim_5_20.pl` is the original Fizzim 5.20 Perl backend pulled from
this repository's history. The current regression flow converts
`testcases/generic_state_machine.fzm` into a Fizzim-1-compatible diagram and
then runs this legacy backend on that converted diagram.

The feature-side RTL is generated by the current `fizzim.pl`. The legacy-side
RTL is generated by `testcases/tb/legacy/fizzim_5_20.pl`. The SystemVerilog
testbench compares the two modules cycle by cycle.

`legacy/FizzimGui_5_20.java` is kept only as a source reference for the old GUI.
It is not compiled or used by the automated test.

Run everything from the repository root:

```bash
make test
```

On Windows without GNU Make:

```powershell
.\make.cmd test
```
